# 🔗 DevLinks (LinkTree-like)

Este é um projeto pessoal de agregação de links, seguindo o conceito de LinkTree, desenvolvido como parte do aprendizado em tecnologias web essenciais. O principal destaque é a funcionalidade de alternância de tema (Dark/Light Mode) implementada de forma dinâmica.

## ✨ Destaques do Projeto

* **Dark/Light Mode:** Funcionalidade completa de alternância de tema, que troca cores, bordas e imagens de fundo dinamicamente.
* **Tecnologias Essenciais:** Implementado utilizando apenas **HTML, CSS e JavaScript Vanilla**.
* **Design Responsivo:** Estrutura otimizada para visualização em dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

| Categoria | Tecnologia | Uso |
| :--- | :--- | :--- |
| **Estrutura** | HTML5 | Semântica e organização do conteúdo. |
| **Estilização** | CSS3 (Custom Properties) | Design responsivo e implementação eficiente do tema (variáveis CSS). |
| **Funcionalidade** | JavaScript (Vanilla) | Lógica para a alternância de classe e troca de imagem/alt text. |
| **Ícones** | IonIcons | Ícones dinâmicos para links sociais. |
| **Design** | Figma | Conceitos de UI/UX |

## ⚙️ Configuração e Instalação

Para rodar o projeto localmente, siga estes passos:

1.  **Clone o Repositório:**
    ```bash
    git clone [Link do seu repositório]
    ```
2.  **Navegue até o Diretório:**
    ```bash
    cd DevLinks
    ```
3.  **Abra o `index.html`:**
    * Simplesmente abra o arquivo `index.html` no seu navegador.
    * **Alternativamente:** Se estiver usando o VS Code, utilize a extensão **Live Server** (recomendada no `extension.json`) para abrir em um servidor local.
